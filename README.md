# Ravenous

위치 기반 비즈니스 정보를 제공하는 Yelp API를 이용한
Create React App 기반 Web App이다.<br><br><br>

## 이용

1. 검색 옵션 설정
2. Business와 Where에 입력 (영문 기반, 예: burger NY)
3. 검색 실행<br><br>

## 구조

```jsx
<Searchbar />   // data fetch in Yelp.js
<BusinessList />
  <Business />
```

<br>

## 참여도

JavaScript 코드는 본인 작성했으며 CSS와 프로젝트 구조는 튜토리얼을 일부 참고하여, React의 기본적인 사용법과 구조를 익힘.
